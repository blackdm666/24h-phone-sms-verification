## 基本使用流程

1. 打开需要验证的目标平台，例如 ChatGPT、Google 或 Telegram。
2. 进入手机号验证页面，确认目标平台已经要求输入手机号。
3. 打开 88FK 对应项目入口。
4. 按页面提示获取手机号。
5. 将手机号填写到目标平台并发送验证码。
6. 回到 88FK 页面查看短信内容。
7. 复制验证码，回到目标平台完成验证。

如果验证码长时间未到，可以按页面提示取消当前号码、切换线路或重新获取号码。不同平台风控策略不同，不建议在目标平台连续频繁提交同一个号码。

## 常见问题

### 收不到验证码怎么办？

先确认目标平台已经点击发送验证码。如果等待后仍未收到，可以取消当前号码，重新获取新号码或切换线路。

### 可以重复使用同一个号码吗？

不同平台规则不同。一般不建议对同一个号码反复提交，尤其是 Google、OpenAI 等风控较严格的平台。

### 为什么有些平台提示号码不可用？

这通常和目标平台风控、号码地区、线路质量或提交频率有关。可以更换线路或稍后重试。
