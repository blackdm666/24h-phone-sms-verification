## 基本使用流程

1. 打开需要验证的目标平台，例如 ChatGPT、Google、Telegram、WhatsApp 或 Discord。
2. 进入目标平台的手机号验证页面。
3. 回到 88FK，选择对应项目入口。
4. 输入卡密并选择线路。
5. 点击获取号码。
6. 把手机号填写到目标平台。
7. 在目标平台点击发送验证码。
8. 回到 88FK 查看短信内容。
9. 复制验证码，回到目标平台完成验证。

如果验证码迟迟不到，可以按页面提示取消当前号码、切换线路或重新获取新号码。不同平台风控策略不一样，不建议在目标平台连续频繁提交同一个号码。
